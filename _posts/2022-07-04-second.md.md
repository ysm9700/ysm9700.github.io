## 데이타 타입
### 정수형(int), 실수형(float), 문자열(str), 논리형(bool)


```python
a = 100
b = 8907.07
c = 'python'
d = True
```


```python
print(a)
print(b)
print(c)
print(d)
```

    100
    8907.07
    python
    True
    

###  type():데이타 타입


```python
print(type(a))
print(type(b))
print(type(c))
print(type(d))
```

    <class 'int'>
    <class 'float'>
    <class 'str'>
    <class 'bool'>
    

### 데이타 타입의 형변환 함수

```python
d = True
```


```python

```


    100 <class 'str'>
    


```python

```

    True <class 'str'>
    


